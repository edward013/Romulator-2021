The interface will be the part on your computer and control the Romulator to react.

Constants.cs contain the constants you need to use will operating the program.exe.

Program.cs contain the main program for interface.

SerialCommunications.cs are the file which achieve the communication between your computer and the board itself.

VerifyRecord.cs verify the file siza, format and code inside for you. Hex file connot be upload due to our code problem, therefore upload a S record is the only choice.

program.exe are the application that form by all four up files, and it needs to be open in the command window in your computer.
